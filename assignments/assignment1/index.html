Overview of the List Assignment

The purpose of this assignment is to give you experience using IntelliJ to develop a simple Java List class. An List is an abstract data type (ADT) with operations such as get(), get(), and size(). Your task is to implement a list of characters in Java.

Graduate students (or students who are taking this course for graduate credit) must do the following additional work:

  • Implement the requirements of the Iterable interface.

As an implementation node, it may be helpful to assume that there is a 'dummy node' present in the list to limit the number of special cases in your code. The dummy node is a node that appears (usually) at the head of the list and is present event if the list is 'empty.'
